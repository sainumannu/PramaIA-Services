## Organizzazione Documentazione

Per mantenere la documentazione organizzata:

1. I nuovi documenti dovrebbero essere aggiunti a questo indice
2. I documenti obsoleti dovrebbero essere spostati nella directory `archivio/`
3. Aggiornare questo indice quando si apportano modifiche significative alla documentazione

## Convenzioni di Denominazione

Per i nuovi documenti:
- Utilizzare MAIUSCOLE e underscore per i nomi dei file
- Includere un prefisso che indichi la categoria (es. PDK_, PLUGIN_, API_)
- Utilizzare suffissi come _REFERENCE, _GUIDE, _TUTORIAL in base al tipo di documento
